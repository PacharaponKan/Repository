## 📚 What I Learned

### 🔹 Day 01: Digital Foundation
- เรียนรู้พื้นฐานระบบดิจิทัล โครงสร้างคอมพิวเตอร์ และหลักการคิดเชิงคำนวณ (Computational Thinking) เพื่อนำมาประยุกต์ใช้ในการแก้ปัญหาอย่างเป็นระบบ

### 🔹 Day 02: Logic & Flowgorithm Lab 1
- ทำความเข้าใจตรรกะการทำงานของโปรแกรมผ่านผังงาน (Flowchart) การใช้เครื่องมือ Flowgorithm และการควบคุมทิศทางแบบมีเงื่อนไข (Conditional Statements / If-Else)

### 🔹 Day 03: Flowgorithm Lab 2–3
- ศึกษาโครงสร้างการทำงานแบบซ้ำ (Looping: For, While) การจัดการเงื่อนไขที่ซับซ้อน (Nested If) และการทดสอบระบบด้วย Test Case เพื่อตรวจสอบความถูกต้องของตรรกะ

### 🔹 Day 04: AI, Web Technology and Network
- เรียนรู้พื้นฐาน Web Technology (HTML, CSS), สถาปัตยกรรมเครือข่ายเบื้องต้น (Client-Server, IP Address) และแนวทางการใช้ AI เพื่อช่วยเพิ่มประสิทธิภาพการทำงาน
